Setup
Pull and run the container image
CLI
docker pull mcr.microsoft.com/mssql/server:2019-latest docker run --name localsql19 -e 'ACCEPT_EULA=Y' -e 'SA_PASSWORD=yourStrong(!)Password' -p 1433:1433 -d mcr.microsoft.com/mssql/server:2019-latest
Bash
Remove your container
docker stop sqlserverdev19 docker rm sqlserverdev19
Bash
Docker Compose
version: "3.2" services: sql-server-db: container_name: sql-server-db image: microsoft/mssql-server-linux:2017-latest ports: - "1433:1433" environment: SA_PASSWORD: "change_this_password" ACCEPT_EULA: "Y" sqldata: image: mcr.microsoft.com/mssql/server:2017-latest environment: - SA_PASSWORD=Pass@word - ACCEPT_EULA=Y ports: - "5434:1433" version: '3.7' services: SQLServer: image: mcr.microsoft.com/mssql/server:2019-latest environment: - ACCEPT_EULA=Y - SA_PASSWORD=@Password1230 ports: - '5434:1433'
YAML
Connect to SQL Server
From Win(Parallel Desktop VM):
User mac ip address from windows since it's bridge over to windows VM so you can use that to connect to the sql server container.
"Server=<mac ip address>;Database=samuraisdb;User Id=sa;Password=@Password1230;MultipleActiveResultSets=true" "Server=10.0.1.20;Database=samuraisdb;User Id=sa;Password=@Password1230;MultipleActiveResultSets=true"
Plain Text
From mac:
Docker is map to localhost so you can use that to connect to the sql server container.
"Server=localhost;Database=samuraisdb;User Id=sa;Password=@Password1230" .UseSqlServer("Server=localhost;Database=samuraisdb;User Id=sa;Password=<YourStrong@Passw0rd>");
